Witht he help of Oomox, I created this GTK theme to match my LXqt/Qt/Qt5/Qt6 theme(s).<br>
Why doesn't GTK have an uninvolved but powerful Ui customization settings system?<br>
IDK, but Oomox is a second close, not realtime, but has a preview, so good enough.<br><br>


<h2>Install</h2>
1.) Copy the "Cherry-Blossom" theme to your /home/(username)/.themes folder<br>
2.) Select Cherry-Blossom in your theme selector <br>
Possibly need to chmod +x some shell scripts, not sure what their purposes are, <br>
they came with the templates part of Oomox and I inspected them, they don't do anything dodgy.<br>
If their -x bit state causes the theme not to work for you, please report it as an issue.

<h2>Tweaks</h2>
<b>For OLED power saving or those who don't want whole-screen flashbang in FireFox:</b><br>
In <code>about:config</code> set the following settings (applies to FireFox 140 era),<br>
layout.css.prefers-color-scheme.content-override = 0				<br>
browser.theme.content-theme = 0							<br>
